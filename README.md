# Duck Duck Race 🦆

A playful browser game: enter participant names, each becomes a duck, and the ducks race
randomly on a 3D water track until one winner is selected.

Built with Vue 3, TypeScript, Three.js, and Pinia.

## Develop

```bash
npm install
npm run dev
```

See [src/README.md](src/README.md) for architecture notes and the race algorithm explanation.
